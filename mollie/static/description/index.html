Pay using Mollie

Configure Mollie Account

Mollie enables your business to receive online payments – made to you from all over the globe – with just a few clicks. This modules allows to make payment using Mollie.

Get Credentials

Login into the dashboard of Mollie.
Go to Website Profiles .
Create a website profile and you will find the credentials here.

Acquirer

On the payment page, select the Mollie Payment Acquirer and click on Pay Now button.

Make Payment

After clicking on Pay Now, you will be redirected to Mollie payment page. Enter the details to make the payment

Order Confirmed

Once the Payment is made the order is confirmed.

Order in Mollie Dashboard

As Mollie does not provide any dashboard, it is not possible to see the test transaction in dashboard.
However live transaction can be checked in dashboard of Mollie.

For test environment, 'EUR' is the only valid currency
